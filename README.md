# Reviewar

ActiveRecordで実施されたSQLをログ出力します。

ほぼ[autodoc](https://github.com/r7kamura)でSQL出力部分だけ手を入れてます。

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'reviewar'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install reviewar
